demo步驟：
1.移動到目錄 dapp-react/client/src/contracts
2.刪除所有的.json檔，回到dapp-react/client
3.執行指令 truffle migrate
4.在metamask匯入並使用ganache中index為0的帳戶
5.執行指令 npm run start
	5.1 若出現permission denied的error，執行指令 cd ./node_modules/.bin/
	5.2 執行指令 chmod u+x react-scripts
	5.3 回到dapp-react/client，繼續下方的步驟
6.點擊跳出視窗的右上方的register
7.在nickname和introduce yourself欄位分別輸入 Mynickname 和 Hi
8.點擊submit並且在metamask中確認送出transaction
9.點擊左上方Create project
10.在Project Title和Project Introduction欄位分別輸入Test Title和Test Intro
11.在Target Money欄位輸入10
12.在Stage Number欄位輸入2
13.在Deadline欄位輸入2020/05/20上午12:00
14.點擊submit並且在metamask中確認送出transaction
15.點擊左上方View project
16.點擊先前創建的project上方的貓咪圖片
17.點擊Sponsor this project
18.在Sponsor Money Amount欄位輸入6
19.點擊submit並且在metamask中確認送出transaction
20.點擊Back to project
21.點擊右下方Start voting並且在metamask中確認送出transaction
22.等transaction成功後點擊右下方load again
23.點擊右下方Agree並且在metamask中確認送出transaction
24.等transaction成功後點擊右下方load again
25.點擊右下方Stop voting並且在metamask中確認送出transaction
26.等transaction成功後點擊右下方load again
27.點擊右下方Start voting並且在metamask中確認送出transaction
28.等transaction成功後點擊右下方load again
29.點擊右下方Disagree並且在metamask中確認送出transaction
30.等transaction成功後點擊右下方load again
31.點擊右下方Stop voting並且在metamask中確認送出transaction
32.等transaction成功後點擊右下方load again
33.在左方Say something欄位中輸入Taiwan No.1
34.點擊submit並且在metamask中確認送出transaction
